# 住所パーサー設計

## 概要

認証APIから取得した住所文字列を解析し、市区町村名を抽出して小選挙区を判定する。

## 住所文字列の形式

デジタル認証アプリAPIが返す住所は、マイナンバーカードの券面記載事項と同じ形式。
例: `千葉県松戸市根本387番地の5`

## パース戦略

日本の住所は以下の構造を持つ:

```
[都道府県] + [市区町村] + [町名以下]
```

### 都道府県パターン
- `東京都`（固定）
- `北海道`（固定）
- `京都府`（固定）
- `大阪府`（固定）
- `{2〜3文字}県`（43県）

### 市区町村パターン（都道府県の後に続く）

1. **政令指定都市の区**: `{市名}{区名}区` — 例: `千葉市中央区`, `さいたま市大宮区`
2. **一般の市**: `{市名}市` — 例: `松戸市`, `つくばみらい市`
3. **東京23区**: `{区名}区` — 例: `千代田区`, `港区`
4. **郡部の町**: `{郡名}郡{町名}町` — 例: `北葛飾郡杉戸町`
5. **郡部の村**: `{郡名}郡{村名}村` — 例: `西多摩郡檜原村`

### 正規表現の設計

単一の正規表現ではエッジケースが多いため、段階的にパースする:

```typescript
export function extractMunicipality(address: string): string | null {
  // Step 1: 都道府県を除去
  const prefecturePattern = /^(?:東京都|北海道|(?:京都|大阪)府|.{2,3}県)/;
  const prefMatch = address.match(prefecturePattern);
  if (!prefMatch) return null;
  const afterPref = address.slice(prefMatch[0].length);

  // Step 2: 政令指定都市の区を検出（最優先）
  // 「さいたま市」「いわき市」など、ひらがな市名にも対応
  const seireichMatch = afterPref.match(/^(.+?市)(.{1,4}区)/);
  if (seireichMatch) {
    // 政令指定都市リストと照合して確認
    const cityName = seireichMatch[1];
    if (SEIREI_SHITEI_CITIES.includes(cityName)) {
      return cityName + seireichMatch[2]; // 例: "千葉市中央区"
    }
  }

  // Step 3: 東京23区（「東京都」直後の「○○区」）
  if (address.startsWith('東京都')) {
    const kuMatch = afterPref.match(/^(.{1,4}区)/);
    if (kuMatch) return kuMatch[1]; // 例: "千代田区"
  }

  // Step 4: 一般の市
  const shiMatch = afterPref.match(/^(.{1,8}?市)/);
  if (shiMatch) return shiMatch[1];

  // Step 5: 郡部（町・村）
  const gunMatch = afterPref.match(/^(.{1,5}郡)(.{1,6}?(?:町|村))/);
  if (gunMatch) return gunMatch[1] + gunMatch[2]; // 例: "北葛飾郡杉戸町"

  return null;
}
```

### 政令指定都市リスト

```typescript
const SEIREI_SHITEI_CITIES = [
  '札幌市', '仙台市', 'さいたま市', '千葉市', '横浜市', '川崎市', '相模原市',
  '新潟市', '静岡市', '浜松市', '名古屋市', '京都市', '大阪市', '堺市',
  '神戸市', '岡山市', '広島市', '北九州市', '福岡市', '熊本市',
];
```

## テストケース

以下のケースをすべてユニットテストでカバーすること:

### 基本パターン
| 入力住所 | 期待される市区町村 | 説明 |
|---|---|---|
| `千葉県松戸市根本387番地の5` | `松戸市` | 一般的な市 |
| `東京都千代田区永田町1丁目7番1号` | `千代田区` | 東京23区 |
| `東京都港区赤坂1丁目2番3号` | `港区` | 東京23区 |
| `千葉県千葉市中央区中央港1丁目` | `千葉市中央区` | 政令指定都市の区 |
| `埼玉県さいたま市大宮区桜木町1丁目7番5号` | `さいたま市大宮区` | ひらがな政令指定都市 |
| `神奈川県横浜市西区みなとみらい2丁目` | `横浜市西区` | 政令指定都市 |

### エッジケース
| 入力住所 | 期待される市区町村 | 説明 |
|---|---|---|
| `茨城県つくばみらい市板橋1234` | `つくばみらい市` | 長いひらがな市名（6文字） |
| `茨城県かすみがうら市上土田500` | `かすみがうら市` | 長いひらがな市名（6文字） |
| `北海道札幌市中央区北1条西2丁目` | `札幌市中央区` | 北海道の政令指定都市 |
| `京都府京都市左京区吉田本町` | `京都市左京区` | 京都の政令指定都市 |
| `大阪府大阪市北区中之島1丁目` | `大阪市北区` | 大阪の政令指定都市 |
| `大阪府堺市堺区南瓦町3番1号` | `堺市堺区` | 堺市（区名が市名と同じ） |
| `千葉県北葛飾郡杉戸町清地2丁目` | `北葛飾郡杉戸町` | 郡部（町） |

| `鹿児島県鹿児島市山下町15番1号` | `鹿児島市` | 県名と市名が同じ |
| `沖縄県うるま市みどり町1丁目1番1号` | `うるま市` | ひらがな市名 |
| `北海道虻田郡ニセコ町字本通105番地` | `虻田郡ニセコ町` | カタカナ町名 |

### 異常系
| 入力住所 | 期待される市区町村 | 説明 |
|---|---|---|
| `松戸市根本387番地` | `null` | 都道府県なし |
| `""` | `null` | 空文字列 |
| `アメリカ合衆国` | `null` | 日本以外 |

## 選挙区マスターデータ

選挙区マスターデータは`data/districts.json`にJSON形式で管理する。

```typescript
// 選挙区マスターの型定義
interface DistrictMaster {
  [districtId: string]: {
    name: string;           // "千葉県第6区"
    prefecture: string;     // "千葉県"
    municipalities: string[]; // ["松戸市"]
  };
}
```

Phase 1では千葉6区と東京1区のみ、Phase 3で全289選挙区を完成させる。
